# ISO 27001 Risk Assessment Lab

## Overview

This repository demonstrates a practical implementation of an Information Security Management System (ISMS) based on ISO/IEC 27001:2022 for a fictional Software-as-a-Service (SaaS) organization.

The project simulates the responsibilities of a Governance, Risk & Compliance (GRC) Analyst during an ISO 27001 implementation, including asset identification, risk assessment, risk treatment planning, and control selection.

---

## Company Scenario

**Organization:** CloudCart Pvt. Ltd.

CloudCart is a fictional SaaS company that provides cloud-based e-commerce solutions for over 500 business customers.

### Company Profile

- Industry: SaaS
- Employees: 60
- Headquarters: Bengaluru, India
- Customers: 500+
- Infrastructure: AWS Cloud
- Productivity Suite: Microsoft 365
- Source Code: GitHub
- Project Management: Jira & Confluence

---

## Project Objectives

- Define the ISMS Scope
- Identify Information Assets
- Perform Risk Assessment
- Develop a Risk Register
- Prepare Risk Treatment Plan
- Map ISO/IEC 27001:2022 Annex A Controls
- Create a Statement of Applicability (SoA)
- Prepare Information Security Documentation
